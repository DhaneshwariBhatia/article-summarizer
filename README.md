# AI Article Summarizer

This project is an AI-powered Article Summarizer built using:

- Hugging Face Transformers
- Gradio

## Features

- Accepts long articles
- Generates concise summaries
- Adjustable summary length using:
  - max_length
  - min_length

## Model Used

facebook/bart-large-cnn

## Run Locally

```bash
pip install -r requirements.txt
python app.py
